le language JAVA

Apparu en 1995, le langage Java a eu beaucoup de succès dans un premier temps grâce à ses "applets java", lui permettant de figurer parmi les premiers à gérer des applications clientes à partir d'un navigateur web (ce qui était impossible en HTML). C'est un véritable langage orienté objet traditionnel, bien structuré, compilé, très différent de JavaScript : Java crée des applications compilées, autonomes, sécurisées qui tournent sur la "Machine Virtuelle" et non des scripts interprétés par le navigateur web. Grâce à la "Machine Virtuelle", ces applications sont indépendantes vis à vis de la plateforme utilisatrice (Windows, Linux, MacOS..). Les applications Android utilisent majoritairement le Java, ce qui lui donne une bonne notoriété parmi les développeurs. Il est également utilisé en entreprise côté serveur et pour des "clients lourds" (logiciels effectuant des traitements complexes et autonomes des données transmises). Il existe donc plusieurs plateformes et dénominations de Java à ne pas confondre, et dont les noms ont pu évoluer au fil du temps que l'on pourrait essayer de résumer ainsi : JSE devenu J2SE (Standard / pour desktops) JEE devenu J2EE (Entreprise / pour serveurs web) JME devenu J2ME (Micro Edition / pour smartphones)
